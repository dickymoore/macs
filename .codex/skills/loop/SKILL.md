# Loop Mode

## Purpose
Tell the controller to keep working without interruption until blocked.

## Usage
Invoke with: `$loop`

## When to Use
- Use when you want the controller to autonomously progress through tasks
- Use when you don't need to be interrupted for routine decisions
- The controller will only break silence when completely blocked or complete
