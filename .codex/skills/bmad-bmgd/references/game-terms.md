# BMGD Terms (summary from docs)

- Game Brief: core vision, pillars, audience, scope; foundation for the GDD.
- GDD: comprehensive game design document (mechanics, systems, content).
- Core Fantasy: emotional experience players seek.
- Core Loop: repeating cycle of actions.
- Design Pillars: 3-5 principles guiding decisions.
- MDA Framework: Mechanics -> Dynamics -> Aesthetics.
- Player Agency: meaningful player choices and impact.
- Meta-Progression: persistence across runs/sessions.
- Procedural Generation, Roguelike, Metroidvania: common genre/system concepts.
