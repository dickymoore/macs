# Contributing to MACS

Thanks for your interest in contributing to MACS (Multi Agent Control System).

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a branch for your changes

## Making Changes

- Keep changes focused and atomic
- Follow existing code style
- Test your changes with the bridge and helper scripts
- Update documentation if needed

## Submitting Changes

1. Push your branch to your fork
2. Open a Pull Request against `main`
3. Describe what your changes do and why

## Reporting Issues

- Check existing issues first
- Include steps to reproduce
- Include your environment (OS, tmux version, Python version)

## Code of Conduct

Be respectful and constructive. We're all here to build useful tools.
